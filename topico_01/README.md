## Diferença entre VM, VPS e Nuvem
- **VM Local:** Corre no nosso próprio computador usando recursos físicos locais.
- **VPS:** Servidor virtualizado num centro de dados remoto com recursos fixos contratados.
- **Nuvem (Cloud):** Infraestrutura de rede flexível e escalável distribuída por vários servidores.
